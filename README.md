# [BALLDONTLIE API](https://docs.balldontlie.io/#introduction)
The balldontliejs.js class can be used to request data from the BALLDONTLIE API.

## Dependencies
- axios
- dotenv
```
npm install axios
npm install dotenv
```

## Examples
The tests directory gives examples of proper usage of the API.
